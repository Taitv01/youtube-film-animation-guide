# Film & Animation YouTube Workflow

// turbo-all

## Step 1: Find Ideas (10 min)
Use ChatGPT/Claude with this prompt:
```
Bạn là chuyên gia YouTube trong lĩnh vực Film & Animation.
Cho tôi 5 ý tưởng video Shorts về series "[SERIES NAME]" với:
- Hook gây tò mò
- Độ dài 30-60s
- Góc nhìn mới
```

## Step 2: Copyright Check (5 min)
- NBC Universal, Netflix Originals = Safe ✅
- Disney, Marvel, Warner Bros = Careful ⚠️
- Check if studio aggressively claims on YouTube

## Step 3: Collect Clips (15 min)
- Use OBS Studio to screen record
- Keep each clip < 30 seconds
- Prefer dialogue scenes over action

## Step 4: Write Script with AI (10 min)
Use ChatGPT/Claude:
```
Scene: [DESCRIPTION]
Write: Hook (3s), Commentary (20s), CTA
Tone: Funny, Gen Z, with emoji
Language: English
```

## Step 5: Edit in CapCut (20 min)
1. Import clip
2. Auto Captions → Montserrat Bold, white, black outline
3. Add zoom effects on reaction faces
4. Use royalty-free music only
5. Export 1080x1920

### Copyright-Safe Checklist:
- [ ] Each clip < 30s continuous
- [ ] Text overlay throughout
- [ ] Background music from free library
- [ ] Original audio < 50% of video

## Step 6: SEO with AI (5 min)
Use ChatGPT:
```
Video about: [DESCRIPTION]
Target: US, 18-35
Write: Title (< 50 chars + emoji), Description, 10 tags
```

## Step 7: Publish (5 min)
Best times for US audience:
- Mon-Thu: 12PM EST = 12AM VN
- Fri: 3PM EST = 3AM VN
- Sat-Sun: 9AM EST = 9PM VN (Saturday night)

## If Copyright Claimed:
Dispute template:
```
This video constitutes fair use under Section 107. 
It is transformative with critical commentary. 
Brief clips (< 30s) with original commentary throughout.
Does not substitute the original work.
```
